## Basic練習課題【初級編解答】

<h3>課題2</h3>

1〜100の数字の時、そのまま数字を出力し、それ以外の数字の時、「範囲外です」と出力してください。

```php
$value2 = -10;
if(0 < $value2 && $value2 <= 100){
	echo $value2;
}else{
	echo "範囲外です";
}
```

<h3>課題4</h3>

2つ変数を用意し、比較して大きい方を出力してください。同じ場合は「同じです」と出力してください。

```php
$x = 3;
$y = 9;
if($x < $y){
	echo $y;
}elseif($y < $x){
	echo $x;
}elseif($y == $x){
	echo "同じです";
}
```
